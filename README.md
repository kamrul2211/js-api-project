# User Management System

A simple **User Management System** built with **HTML, Bootstrap 5, and JavaScript**. This project fetches user data from the JSONPlaceholder API and displays it in a responsive table.

## ✨ Features

- Responsive Bootstrap Navbar
- Hero Section
- Fetch user data using JavaScript Fetch API
- Display users in a Bootstrap Table
- Serial Number
- Edit Button (UI)
- Delete Button (UI)

## 📖 API Used

This project uses the free JSONPlaceholder API.

API Endpoint:

```
https://jsonplaceholder.typicode.com/users
```

---

## ⚙️ How It Works

1. JavaScript sends a request using the Fetch API.
2. User data is received from JSONPlaceholder.
3. The data is displayed dynamically inside the table.
4. Bootstrap is used for responsive UI styling.

## 💻 Installation

Clone the repository


```bash

```
